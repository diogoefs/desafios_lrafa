Machine Learning (SageMaker, Bedrock, etc.)

Tópicos Essenciais

1. Fundamentos da Computação em Nuvem

O que é computação em nuvem?

Seis vantagens da computação em nuvem

Modelos de implantação (Nuvem, Privada, Híbrida)

Infraestrutura global (Regiões e Zonas de Disponibilidade)

2. Computação

Amazon EC2 – Máquinas virtuais sob demanda.

EC2 Auto Scaling – Ajuste automático de capacidade.

Elastic Load Balancing – Distribuição de tráfego.

AWS Lambda – Serverless e execução sob demanda.

3. Armazenamento

Amazon S3 – Armazenamento de objetos.

Amazon Elastic Block Store (EBS) – Armazenamento em bloco.

Amazon Elastic File System (EFS) – Armazenamento de arquivos.

AWS Backup – Estratégias de backup e recuperação.

4. Redes e Conectividade

Amazon VPC – Redes privadas virtuais.

Amazon Route 53 – DNS gerenciado.

Amazon CloudFront – CDN para distribuição de conteúdo.

AWS Direct Connect / VPN – Conectividade híbrida.

5. Bancos de Dados

Amazon RDS – Bancos relacionais gerenciados.

Amazon DynamoDB – Banco NoSQL.

Amazon Aurora – Banco relacional otimizado.

6. Segurança e Conformidade

IAM (Identity and Access Management) – Controle de identidade e permissões.

AWS KMS / Secrets Manager – Gerenciamento de chaves e segredos.

AWS Shield / WAF – Proteção contra ataques.

CloudTrail / CloudWatch – Auditoria e monitoramento.

7. Automação e DevOps

AWS CloudFormation – Infraestrutura como código.

AWS CodePipeline / CodeBuild / CodeDeploy – CI/CD.

AWS Systems Manager – Gerenciamento de instâncias.
